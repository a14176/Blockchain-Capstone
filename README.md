# Blockchain Capstone Project
Using the ERC-721 Non-Fungible Token Standard Contract, this project uses the NFT to assign ownership of a property title. Proof of ownership prior to successful minting of the property title NFT is achieved using a zk-SNARKs based contract that is generated from the Zokrates implementation (Verifier.sol).

1. The Verifer and SolnSquareVerifier contracts are deployed to the Rinkeby network.
2. The mint.js script is used to mint 10 NFT properties on Rinkeby.
3. An Opensea collection shows these 10 NFT properties (using the SolnSquareVerifier contract address).
4. Five of the NFT properties have been purchased on Opensea via a different account.

## Installation
```shell
> npm install
> touch ./eth-contracts/.secret
```
I used the local truffle for compile and testing which means .secret wasn't neccessary until I deployed to the Rinkeby network. 
Append your wallet secret key(words phrase) into `.secret` 

## Deployment

```shell
> cd eth-contracts
> truffle develop
> compile
> test
```
when ready to deploy to Rinkeby
```shell
> migrate --network rinkeby
```

or
```shell
> truffle compile && truffle test && truffle migrate --network rinkeby
```
## Implement Zokrates using a docker container

```shell
> docker run -v ./zokrates/code/square:/home/zokrates/code -ti zokrates/zokrates /bin/bash

> zokrates compile -i square.code
> zokrates setup
> zokrates compute-witness -a 3 9
> zokrates generate-proof
> zokrates export-verifier
```
----------------------------------------------------------------
## Minting NFT's on the Rinkeby network
To generate 10 proofs, (each is used within the mint process), this must be run within the docker environment from above
```bash
mkdir proofs

for i in `seq 10 19`; do
  zokrates compute-witness -o proofs/witness.$i -a $i $((i*i))
  zokrates generate-proof -j proofs/proof-$i.json -w proofs/witness.$i
done
```
To mint 10 NFT's use the mint.js script. This requires a .env file containing the following export parameters:

```bash
export INFURA_KEY=""
export MNEMONIC=""
export OWNER_ADDRESS="0xeB50058CfFDF8f25830884fac1F890E479BbfFa4"
export NFT_CONTRACT_ADDRESS="0x30758d5015F0e786671F51Fc53FDe96f11f22568"
export NETWORK="rinkeby"

source .env
node mint.js
```
